# HireGold - Premium Job Board

A modern, elegant job board application with a sleek black and gold design.

## Features

- Browse job listings
- Search jobs by title or company
- Filter by job type (Full-time, Part-time, Contract, Remote)
- Apply to jobs directly
- Responsive design

## Tech Stack

- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Backend**: Node.js, Express
- **Styling**: Custom CSS with Inter font

## Local Development

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open browser to `http://localhost:3000`

## Deployment

This app is deployed on Railway and accessible at your live URL.

## API Endpoints

- `GET /api/jobs` - Get all job listings
- `POST /api/apply` - Submit job application
